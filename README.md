# 💰 FinTrack

**FinTrack** is a web-based personal finance management system that helps users track expenses, manage budgets, and gain insights into their spending habits. Built using **HTML, CSS, JavaScript, PHP, and MySQL**, it provides a simple yet powerful interface for financial tracking.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 💸 Add, Edit, Delete Expenses
* 📊 Budget Management
* 📅 Date-wise Expense Tracking
* 📂 Category-based Organization
* 🔍 Search & Filter Transactions
* 📈 Dashboard with Financial Overview
* 📱 Responsive Design

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server:** Apache (XAMPP / WAMP / LAMP)

---

## ⚙️ Installation

1. Clone the repository
   git clone https://github.com/Samar2442/fin-track.git

2. Move to server directory
   C:\xampp\htdocs\FinTrack

3. Start Apache & MySQL

4. Create Database

   * Open phpMyAdmin
   * Create database: fintrack
   * Import database.sql

5. Configure Database
   Open config/db.php and update:

   $host = "localhost";
   $user = "root";
   $password = "";
   $database = "fintrack";

6. Run the project
   http://localhost/FinTrack
